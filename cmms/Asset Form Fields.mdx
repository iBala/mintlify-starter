This page details all the default Form Fields that are available with an Asset

# Asset Form Fields
| Field    | Description |
| -------- | ------- |
| Code  | Unique code of the asset. By default, this is auto generated by ZORP. You can change it to your custom code |
| Name | Name of the Asset     |
| Asset Timeline    | Asset timelines gives a snapshot of the Asset's uptime over the last 10 days. This is what the colors indicate.<br> Green - Operational<br>Red - Non-operational<br>Blue - Maintenance going on but operational    |
| Image | Image of the Asset |
| Status | Status of the Asset. By default, the following statuses are available <br>- Operational<br>- Not Operational<br>You can add custom Statuses based on your processes. |
| Location | Link to the location where this Asset is located |
| Purchase Date | Date when the asset is purchased |
| Purchase Price | Price of the Asset when it is purchased |
| Warranty Expiration | Date on which the Asset warranty gets expired |
| Work Orders | All the work orders associated with this Asset |
| Asset Type | Type of the Asset. This is useful for grouping assets by type while reporting. Default options are Machinery, Equipment, Tools, Furniture, IT Infrastructure, Specialized Equipment, Consumables. You can add custom types based on your requirements. |
| Criticality | Criticality of the Asset. Default options are Critical, Important, Normal. This is useful for grouping while reporting. |
| Useful Life | Number of months the asset is useful before scraping or selling it off |
| Residual Value | Value of the Asset at the end of useful life |
| Depreciation Method | Type of depreciation to apply on this Asset. Following types are supported. <br>- Straight Line<br>- Declining Balance<br>- Double Declining Balance<br>- Sum of the Year Digits |
