# Asset Form Fields

- **Code**: 
  - Unique code of the asset. 
  - Auto-generated by ERPLite by default.
  - Custom codes can be assigned.

- **Name**: 
  - Name of the Asset.

- **Asset Timeline**: 
  - Snapshot of the Asset's uptime over the last 10 days.
  - Green: Operational
  - Red: Non-operational
  - Blue: Maintenance but operational

- **Image**: 
  - Image of the Asset.

- **Status**: 
  - Status of the Asset.
  - Default statuses: Operational, Not Operational.
  - Custom statuses can be added based on processes.

- **Location**: 
  - Link to the location where the Asset is located.

- **Purchase Date**: 
  - Date when the asset was purchased.

- **Purchase Price**: 
  - Price of the Asset at the time of purchase.

- **Warranty Expiration**: 
  - Date on which the Asset warranty expires.

- **Work Orders**: 
  - All the work orders associated with this Asset.

- **Asset Type**: 
  - Type of the Asset.
  - Useful for grouping assets by type while reporting.
  - Default options: Machinery, Equipment, Tools, Furniture, IT Infrastructure, Specialized Equipment, Consumables.
  - Custom types can be added.

- **Criticality**: 
  - Criticality of the Asset.
  - Default options: Critical, Important, Normal.
  - Useful for grouping while reporting.

- **Useful Life**: 
  - Number of months the asset is useful before being scrapped or sold.

- **Residual Value**: 
  - Value of the Asset at the end of its useful life.

- **Depreciation Method**: 
  - Type of depreciation to apply to this Asset.
  - Supported methods: Straight Line, Declining Balance, Double Declining Balance, Sum of the Year Digits.
