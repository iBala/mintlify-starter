The Scan component can be used to scan either QR code or Barcode from the Zorp app.


## Setting up a Scan Component

  

1. Drag and Drop the Scan component onto the App Screen builder of the canvas.

2. Select the Datafield where you want to store the scan values

>[Click here](/Getting%20Started/Creating%20Datafields) to learn about Data Fields

3. Enter the Title to describe the scan being done.

4. Add a default value if required

5. Specify the properties like minimum and maximum scans needed

  
  

## Where can I use this?

  

1. Product scan

2. Warehousing/delivery
