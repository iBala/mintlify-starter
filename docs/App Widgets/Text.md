# Text

  

The Text component is a display Component where you can show any Text on the App. This is one of the most basic and useful components for variety of use cases

<img src="https://zorp-assets.s3.ap-south-1.amazonaws.com/docs/text.gif" alt="Zorp TEXT component"/>

## Setting up a Text Component


1. Drag and Drop the Text component onto the App Screen builder of the canvas.

	The text component has
	- Label
	- Main Text
	- Subtext

2. Each one of these 3 fields can be used to display the text you would want the user to see on the mobile app.

>You can provide the actual values for these texts or give a dynamic value which can refer to the task's data fields. Refer to the [Using data fields](https://docs.zorp.one)  

3. Text component provides the capability 
	- to choose a different color for the text
	- Different size of the text
	- Visibility of the text
