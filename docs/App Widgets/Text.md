# Text

The Text component is a display Component where you can show any Text on the App.

## Setting up a Text Component

1. Drag and Drop the Text component onto the App Screen builder of the canvas.
    The text component has 
    - Label
    - Text
    - Subtext 

Each one of these 3 fields can be used to display the text you would want the user to see on the mobile app. 
>You can hardcode these texts or give a dynamic value which would be task specific from the fields on the Text component.
>Cick here to know how to create a Data field for a text component and how to refer the datafield values.
3. Enter the Title to describe the Text component being used.
