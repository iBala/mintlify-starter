# Map

The Map component is a display Component where you can show a location on the App in a Map view which the user can use to navigate and reach the location.

## Setting up a Map Component

1. Drag and Drop the Map component onto the App Screen builder of the canvas.
    The map component needs 
    - Latitude and
    - Longitude to diplay the location. 
>You can hardcode these lat long values or give a dynamic value which would be task specific from the Datafields.
>[Click here](/Getting%20Started/Creating%20Datafields) to learn about Data Fields
3. Enter the Title to describe the Map component being used.
