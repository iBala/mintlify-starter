# Text Input

  

The Text Input component is a very common component which can be used to input any text.

  <img  src="https://zorp-assets.s3.ap-south-1.amazonaws.com/docs/textinput.gif"  alt="Zorp TEXT Input component"/>

## Setting up a Text Input Component

  

1. Drag and Drop the Text Input component onto the App Screen builder of the canvas.

2. Select the Datafield where you want to store the Text Input from the component.

>Click here to learn how to create a Datafield for a Text input component

3. Enter the Title to describe the Text Input being captured.

4. Add a default value for the Text Input

5. In case the text input is mandatory, choose  `Is Required` property

  
  

## Where can I use this?

  

1. All Data collection use cases.

2. Ticketing system app
