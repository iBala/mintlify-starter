# Testing your Workflow

To test your Workflow you need to:
1. Create a Task 
2. Execute the task on the Zorp Mobile App.

## Creating a Task

There are multiple ways of creating a task. 
- Single task creation
- Bulk task creation
- Task creation using API

## Using the Zorp Mobile App

Once your Workflow is ready and you have created a Task you can start testing your Workflow by Installing the Zorp app from Playstore or AppStore.

[Playstore](https://bit.ly/3xzCwZT)
[Appstore](https://apple.co/3Eme2om)

1. To begin testing your Workflow, login to the Zorp App using the User credentials created in Users. 
2. Upon Login Zorp tasks you to the Home screen where you see all your tasks represented as a Task card. 
3. Click on the Task of your Workflow and start executing it.
