### 4. Editing the permissions of the Role

Click on the role you had just created


### 5. Click on EDIT

<kbd>
  ![Click on Edit](https://images.tango.us/public/screenshot_9876d614-1dd0-4f52-bca0-5c5ac0cffa33?crop=focalpoint&fit=crop&fp-x=0.9752&fp-y=0.0984&fp-z=2.9554&w=600&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)
</kbd>


### 6. Check the permissions that you want to provide to the role
Create: The user would be able to create the task/user/team

Edit: The user has access to edit the configuration or tasks.

View: The user can View the tasks/users/teams

Remove: The user will be able to make the configurations inactive  
  
Note: View and Edit for Tasks is mandatory for the role to be able to execute tasks on the app.
<kbd>
  ![Set mandatory permissions](https://images.tango.us/public/screenshot_16f1ae95-244c-4151-bc90-23ae808df09c?crop=focalpoint&fit=crop&fp-x=0.1087&fp-y=0.1364&fp-z=2.8878&w=600&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)
</kbd>


### 7. Click on EDIT ROLE to Save the changes

<kbd>
  ![Save the changes](https://images.tango.us/public/screenshot_fa056c2b-8c63-48b2-82b4-4b399cdc6b00?crop=focalpoint&fit=crop&fp-x=0.8736&fp-y=0.9455&fp-z=4.0000&w=600&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)

Your new role is ready. You can set the role of a user in the user tab. The user will be able to use the ZORP apps based on the permissions set.
