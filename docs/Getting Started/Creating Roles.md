# [Roles](https://controlcenter.dev.zorp.one/#/workflows)
Roles control the way an user can use the features and tasks in the Zorp ecosystem.


## Creating a New role
Click on Roles on the Sidebar

### 1. Click on NEW ROLE

![Step 3 screenshot](https://images.tango.us/public/screenshot_3788b38f-5979-4216-a0c7-e6bdea9ba362?crop=focalpoint&fit=crop&fp-x=0.9696&fp-y=0.0977&fp-z=3.0772&w=1200&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)


### 2. Type the Name of the Role and the description.

![Step 4 screenshot](https://images.tango.us/public/screenshot_b5d39603-32c3-4b58-a282-fadb289a172c?crop=focalpoint&fit=crop&fp-x=0.5233&fp-y=0.1982&fp-z=1.6327&w=1200&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)


### 3. Click on CREATE ROLE

![Step 5 screenshot](https://images.tango.us/public/screenshot_e6b5516d-d610-42f3-945b-64e0357316af?crop=focalpoint&fit=crop&fp-x=0.4045&fp-y=0.3949&fp-z=2.7586&w=1200&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)


### 4. Editing the permissions of the Role

Click on the role you had just created


### 5. Click on EDIT

![Step 7 screenshot](https://images.tango.us/public/screenshot_9876d614-1dd0-4f52-bca0-5c5ac0cffa33?crop=focalpoint&fit=crop&fp-x=0.9752&fp-y=0.0984&fp-z=2.9554&w=1200&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)


### 6. Check the permissions that you want to provide to the role
Create: The user would be able to create the task/user/team

Edit: The user has access to edit the configuration or tasks.

View: The user can View the tasks/users/teams

Remove: The user will be able to make the configurations inactive  
  
Note: View and Edit for Tasks is mandatory for the role to be able to execute tasks on the app.
![Step 8 screenshot](https://images.tango.us/public/screenshot_16f1ae95-244c-4151-bc90-23ae808df09c?crop=focalpoint&fit=crop&fp-x=0.1087&fp-y=0.1364&fp-z=2.8878&w=1200&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)


### 7. Click on EDIT ROLE to Save the changes

![Step 9 screenshot](https://images.tango.us/public/screenshot_fa056c2b-8c63-48b2-82b4-4b399cdc6b00?crop=focalpoint&fit=crop&fp-x=0.8736&fp-y=0.9455&fp-z=4.0000&w=1200&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)
