### [Users](https://controlcenter.dev.zorp.one/#/roles)
A User needs to be created from an Account on Zorp for him/her to be able to login from the Mobile/Web and execute the workflow or edit the workflow and the configurations.


### 1. Click on Users
<kbd>![Step 2 screenshot](https://images.tango.us/public/screenshot_51dd81fb-513f-491c-95be-4ed80abb5725?crop=focalpoint&fit=crop&fp-x=0.0203&fp-y=0.1733&fp-z=3.0318&w=600&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)</kbd>


### 2. Click on NEW USER
<kbd>![Step 3 screenshot](https://images.tango.us/public/screenshot_ac4878ba-8f03-444d-808d-7a604e4e3949?crop=focalpoint&fit=crop&fp-x=0.9530&fp-y=0.0984&fp-z=2.9554&w=600&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)</kbd>


### 3. Type the Name and Mobile number for the User.
<kbd>![Step 4 screenshot](https://images.tango.us/public/screenshot_c569d1c9-6c4f-43b6-a472-6f375d67093d?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&fp-z=1.0000&w=600&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)</kbd>


### 4. Select the Role for the User
<kbd>![Step 5 screenshot](https://images.tango.us/public/screenshot_fe9585e2-9f15-4740-a753-51146cf52dcf?crop=focalpoint&fit=crop&fp-x=0.7774&fp-y=0.2680&fp-z=2.9822&w=600&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)</kbd>


### 5. Assign the User to the respective Team
<kbd>![Step 6 screenshot](https://images.tango.us/public/screenshot_a52717bc-dd33-4ad5-830d-3f903ccc31c7?crop=focalpoint&fit=crop&fp-x=0.8337&fp-y=0.2996&fp-z=2.9822&w=600&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)</kbd>


### 6. Set the password for the user
<kbd>![Step 7 screenshot](https://images.tango.us/public/screenshot_3f91b712-6116-4b41-8905-116514b1cc68?crop=focalpoint&fit=crop&fp-x=0.3049&fp-y=0.3675&fp-z=2.0513&w=600&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)</kbd>


### 7. Enter the emailId of the user if available
<kbd>![Step 8 screenshot](https://images.tango.us/public/screenshot_1ddaa6c3-4506-41a1-a4a2-a20fe823d9b6?crop=focalpoint&fit=crop&fp-x=0.5219&fp-y=0.4695&fp-z=1.0519&w=600&mark-w=0.2&mark-pad=0&mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&ar=2880%3A1642)</kbd>


### 8. Click on CREATE USER.
This sends out a text message to the New user with his login credentials and app store and playstore links.
